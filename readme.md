# C++ Tip Calculator

### Description

This is a simple tip calculator that take a user's inputed bill amount. After user submits their bill amount the user is then prompted to to input their desired tip amount. The program will then print the desired tip amount rounded to the nearest whole number.

## Instructions

in your terminal run the following commands:

```bash
g++ tip.cpp -o tip

```

once compiled run the following command:

```bash
./tip
```
